# CloudModular

## Backend

### Installation
#### .env
```
SERVER_PORT=<서버가 돌아갈 포트>
SERVER_HOST=<서버가 돌아갈 호스트(프록시 서버를 사용하지 않을 때 활용)>
SERVER_STORAGE=<서버에 돌아가는 데이터들을 저장할 때 사용>

DB_TYPE=<mysql/mariadb of sqlite>
DB_HOST=<DB Host>
DB_PORT=<DB Port>
DB_DATABASE=<DB Name>
DB_USER=<DB User ID>
DB_PASSWD=<DB User pswd>

ADMIN_EMAIL=<관리자 이메일>
ADMIN_EMAIL_PASSWD=<관리자 2차 이메일 패스워드>
```
